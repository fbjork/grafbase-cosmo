# Grafbase + Cosmo example

# Getting Started

# Install and run Cosmo
Follow these steps:
https://github.com/wundergraph/cosmo#getting-started

# Configure the Cosmo data source

Set the following environment variables:

```
COSMO_API_URL=
COSMO_API_KEY=
```

# Run Grafbase

```
npx grafbase@latest dev
```
